Sensors for Hydroponics with Node-RED

Below are some commonly used sensors in hydroponics, their functions, and a step-by-step explanation on how to integrate them with Node-RED and an ESP32 microcontroller.

1. pH Sensor

What it does: This sensor measures how acidic or basic the water is. Plants need the right pH level to absorb nutrients properly.

How to connect:

  1. Find the pH sensor and its probe.
  2. Connect the sensor's signal output wire to an analog pin on the ESP32.
  3. Power the sensor by connecting the VCC wire to 3.3V or 5V on the ESP32 and the GND wire to the ground.

How to use with Node-RED:

  1. Set up an MQTT connection between the ESP32 and Node-RED.
  2. Read the pH sensor values from the ESP32 and send them via MQTT.
  3. In Node-RED, use a gauge node to display the pH values on a dashboard.
  4. Set up alerts if the pH goes out of the ideal range.

2. EC Sensor

What it does: This sensor measures the electrical conductivity (EC) of water to determine how many nutrients are present.

How to connect:

  1. Attach the EC sensor probe to its signal board.
  2. Connect the signal wire to an analog input on the ESP32.
  3. Power the EC sensor by connecting it to 3.3V/5V and ground.

How to use with Node-RED:

  1. Program the ESP32 to read EC values and send them to Node-RED via MQTT.
  2. In Node-RED, create a visual graph to monitor changes over time.
  3. Use function nodes to send alerts if the EC level is too high or too low.

3. Temperature & Humidity Sensor (DHT22)

What it does: Measures air temperature and humidity levels in the hydroponics environment.

How to connect:

  1. Connect the DHT22 sensor’s data pin to a digital pin on the ESP32.
  2. Provide power by connecting the VCC pin to 3.3V/5V and GND to ground.

How to use with Node-RED:

  1. Use an ESP32 program to read temperature and humidity values.
  2. Send these values to Node-RED using MQTT.
  3. Use a chart node in Node-RED to display the temperature and humidity trends.
  4. Trigger notifications when conditions go beyond ideal growing ranges.

4. Water Level Sensor

What it does: Ensures that the hydroponic system has enough water.

How to connect:

  1. Attach the water level sensor to the side of the water tank.
  2. Connect its signal wire to an analog or digital pin on the ESP32.
  3. Power the sensor by connecting it to 3.3V/5V and ground.

How to use with Node-RED:

  1. Program the ESP32 to read water level data and send it to Node-RED.
  2. Use a bar graph in Node-RED to show water levels.
  3. Set up an automated water pump control when levels get too low.

5. Light Sensor

What it does: Measures light intensity to ensure plants receive the correct amount of light.

How to connect:

  1. Connect the light sensor’s signal output to an analog pin on the ESP32.
  2. Power the sensor with 3.3V/5V and connect ground.

How to use with Node-RED:

  1. Read sensor values from the ESP32 and send them via MQTT.
  2. In Node-RED, display light intensity using a gauge.
  3. Use a function node to trigger LED grow lights when natural light is too low.
